# Промпт для дизайнера (claude.ai/design)

Скопируйте текст ниже целиком в https://claude.ai/design.
Рекомендую делать в два захода: сначала витрина, потом админка (второй промпт в конце).

---

## Промпт 1 — Витрина

Спроектируй интернет-магазин автомобильных запчастей **Sacramento** (Молдова,
язык интерфейса — русский, валюта — лей, «MDL»). Десктоп + мобильная версия.
Дизайн-система — Material (MUI), но без «дефолтного» вида Material.

**Бренд и настроение.** Логотипа нет — придумай: словесный знак «SACRAMENTO»,
технологичный, можно с мотивом детали/шестерни/шеврона. Характер: надёжный
гараж-профи, а не глянцевый маркетплейс. Палитра: тёмный графит/антрацит как
основа, один энергичный акцент (предложи 2–3 варианта: сигнальный оранжевый,
лаймово-жёлтый, электрик-синий), много белого в контентных зонах. Крупная
моноширинная типографика для артикулов и цен. Сетка плотная, инженерная.

**Ключевая фича — подбор по VIN.** На главной — hero-блок с большим полем ввода
VIN (17 символов) и слоганом вида «Введите VIN — покажем запчасти именно для
вашего авто». После ввода: шаг подтверждения «Ваш автомобиль: Audi A4, 2016 —
уточните модель» (выбор из 2–4 карточек-вариантов), затем закреплённый бейдж
автомобиля в шапке сайта с кнопкой «сменить авто». Альтернатива VIN — ручной
подбор: марка → модель → год.

**Страницы витрины:**
1. **Главная** — шапка (лого, поиск по названию/артикулу, корзина, телефон),
   hero с VIN-полем, плитка популярных категорий (радиаторы, оптика, бамперы,
   подкрылки…), преимущества (в наличии на складе в Кишинёве, самовывоз сегодня,
   доставка курьером), подвал с контактами и часами работы.
2. **Каталог/категория** — фильтры слева (категория, бренд, наличие, цена),
   сетка карточек товара: фото, название, артикул моноширинным, цена в леях,
   бейдж «В наличии: 3 шт», бейдж «✓ Подходит для вашего Audi A4» если выбран
   автомобиль, кнопка «В корзину». Переключатель «только в наличии».
3. **Карточка товара** — галерея фото, артикул и OEM-номера (копируются в один
   клик), цена крупно, наличие, применимость («Подходит для: Audi A4 1997–2001,
   Audi A6 1994–1997»), описание, кнопка «В корзину», блок «аналоги/похожие».
4. **Корзина** — список позиций, изменение количества, итого.
5. **Оформление заказа** — один экран или степпер: контакты (имя, телефон,
   e-mail опционально) → получение: «Курьер — 50 MDL» / «Самовывоз — бесплатно,
   адрес и часы» → оплата: для курьера «наличными курьеру», для самовывоза
   «картой на месте / наличными» → комментарий → кнопка «Оформить заказ».
   Без регистрации и личного кабинета.
6. **Заказ принят** — номер заказа, что дальше («менеджер позвонит для
   подтверждения»).

**Состояния, которые надо показать:** VIN не распознан (мягкий фолбэк на ручной
выбор авто), пустая выдача («для вашего авто пока ничего нет в наличии — позвоните
нам»), товар закончился, ошибка валидации VIN, мобильная корзина.

## Промпт 2 — Админка

Спроектируй админ-панель интернет-магазина автозапчастей **Sacramento**
(русский язык, MUI). Светлая, сдержанная, плотная информационно — рабочий
инструмент одного администратора. Слева — узкий сайдбар навигации, сверху —
бар со счётчиком новых заказов (бейдж).

**Экраны:**
1. **Dashboard** — новые заказы (карточки со статусом NEW), заказы за сегодня,
   товары с нулевым остатком, текущий курс USD→MDL с пометкой «курс НБМ /
   ручной».
2. **Товары** — таблица: фото-миниатюра, артикул, название, категория, закупка,
   розница, опт, остаток, полка, активность; быстрый поиск, фильтры; кнопка
   «Добавить товар».
3. **Редактор товара** — вкладки: «Основное» (артикул, название, бренд,
   категория, описание, OEM-номера тегами), «Цены» (закупка + валюта USD/MDL,
   наценка или ручная цена, опт), «Фото» (drag&drop, выбор главного), 
   «Применимость» (привязанные автомобили + добавление), «Склад» (остаток, полка).
4. **Заказы** — таблица с фильтром по статусу (чипы: Новый/Подтверждён/Сборка/
   Доставка/Готов к выдаче/Завершён/Отменён), карточка заказа: состав, цены,
   контакты клиента, способ получения и оплаты, смена статуса, переключатель
   «розница / опт» (пересчитывает цены позиций), ручная скидка (% или сумма),
   печать.
5. **Курс и цены** — текущий курс, переключатель «НБМ / вручную», история курса,
   глобальная наценка %, правило округления.
6. **Импорт/экспорт** — история ежедневных снэпшотов (дата, файл CSV/XLSX,
   скачать), кнопка «Выгрузить сейчас»; зона импорта: drag&drop файла →
   экран предпросмотра «Создаётся: 12 · Обновляется: 4870 · Ошибки: 3 (строки
   17, 254, 1003)» → подтверждение → отчёт.
7. **Применимость** — справочник автомобилей (марка/модель/годы) и очередь
   «нераспознанные товары» для массовой привязки.
8. **Настройки** — стоимость доставки курьером, адрес/часы точки самовывоза,
   параметры фото, смена пароля.

**Состояния:** предпросмотр импорта с ошибками, пустой dashboard, подтверждение
смены статуса заказа с резервированием остатков.
